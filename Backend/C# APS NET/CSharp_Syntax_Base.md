# C# - Sintaxe Base

## O que é C#
C# é uma linguagem orientada a objetos utilizada dentro da plataforma .NET.

## Estrutura de Programa
Um programa em C# é composto por:
- Namespaces
- Classes
- Métodos
- Atributos e Propriedades

## Tipos de Dados
- Tipos primitivos (int, float, bool, char)
- Tipos por referência (string, objetos)
- Tipos estruturados (struct, enum)

## Conceitos da Linguagem
- Variáveis e escopo
- Conversão implícita e explícita
- Valor vs referência
- Null e Nullable

## Paradigma
- Total suporte a Programação Orientada a Objetos (POO)
