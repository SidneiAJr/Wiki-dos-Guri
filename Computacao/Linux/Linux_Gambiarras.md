# Linux — Gambiarras para Jogos

Para jogar no Linux, geralmente é preciso recorrer a algumas soluções alternativas como **Lutris** e **WineHQ**.

## WineHQ
O **WineHQ** é útil, mas pode ser trabalhoso:  
é preciso sempre verificar qual versão está instalada, pois versões incompatíveis podem **quebrar jogos ou aplicativos**.

## Lutris
O **Lutris** costuma funcionar bem em várias distribuições,  
mas em outras — como no **MX Linux** — o desempenho pode ser inconsistente ou apresentar falhas.

## Limitações
Alguns jogos ainda estão disponíveis **apenas para Windows**, sem suporte oficial no Linux.

Por isso, às vezes a solução mais prática acaba sendo manter um **dual boot** entre Linux e Windows para garantir compatibilidade total.
