# Linux | WineHQ - O que é?

O **WineHQ** é uma ferramenta que permite rodar **aplicativos do Windows** no Linux, criando uma camada de compatibilidade entre os dois sistemas operacionais.

- **Link do site oficial**: [WineHQ](https://www.winehq.org/)

Com o **Wine**, é possível rodar muitos aplicativos e programas nativos do Windows no Linux, incluindo jogos, mas é importante saber que a compatibilidade nem sempre é perfeita.  
Muitas vezes, os aplicativos funcionam, mas **de forma instável ou com falhas** — ou seja, **funciona meia boca**.

## Principais Desvantagens:
- Pode exigir ajustes na configuração para funcionar corretamente.
- Desempenho nem sempre é o ideal, especialmente para jogos mais pesados.
- Alguns programas não funcionam corretamente ou têm falhas gráficas.

Apesar disso, o Wine ainda é uma opção interessante para quem precisa rodar algo do Windows no Linux sem precisar usar uma máquina virtual ou dual boot.
