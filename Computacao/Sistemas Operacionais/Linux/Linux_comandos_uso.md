# Linux | Lista de Comandos (Uso Pessoal)

Comandos que uso no meu dia a dia no Linux:

- `sudo apt update` — atualiza a lista de pacotes disponíveis  
- `sudo apt upgrade` — atualiza os pacotes instalados  
- `sudo apt dist-upgrade` — atualiza pacotes e lida com dependências complexas  
- `sudo apt autoremove` — remove pacotes desnecessários  

- `vim nomearquivo.sh` — abre/cria um arquivo de script no Vim  
- `echo "mensagem"` — imprime mensagens no terminal  

- `chmod +x nomearquivo.sh` — dá permissão de execução a um arquivo  
- `chmod 777 nomearquivo` — dá permissão total a um arquivo ou pasta  

- `ls` — lista arquivos no diretório atual  
- `ls -a` — lista todos os arquivos, inclusive ocultos  
- `ls -d` — mostra diretórios sem listar conteúdo  
- `pwd` — mostra o diretório atual  

- `sudo kill PID` — finaliza um processo pelo ID  
- `htop` — monitor de processos interativo  
- `sudo` — executa comandos como superusuário
