# .NET - Introdução

## O que é .NET
.NET é uma plataforma de desenvolvimento criada pela Microsoft para construir aplicações de diferentes tipos, como:
- Aplicações Web
- Aplicações Desktop
- APIs
- Serviços
- Aplicações em Nuvem

## Características Principais
- Multiplataforma: Windows, Linux, MacOS
- Linguagens compatíveis: C#, F#, VB.NET
- Gerenciamento automático de memória
- Grande biblioteca padrão

## Versões do .NET
- .NET Framework: Versão antiga e exclusiva para Windows
- .NET Core: Versão reescrita, moderna e multiplataforma
- .NET (ou .NET 5+): Unificação do Framework e Core

## Onde é utilizado
- Desenvolvimento backend
- Sistemas empresariais
- APIs REST
- Aplicações de alto desempenho

## Ecossistema
- ASP.NET Core → Desenvolvimento Web e API
- Entity Framework Core → Acesso a banco de dados
- Blazor → Aplicações web interativas
