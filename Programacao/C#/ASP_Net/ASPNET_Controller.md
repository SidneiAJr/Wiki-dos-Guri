# ASP.NET Core - Controllers

## O que é um Controller
Componente responsável por:
- Receber requisições
- Processar dados ou chamar serviços
- Retornar respostas

## Estrutura
Um controller é composto por:
- Métodos públicos chamados "Ações"
- Retornos que podem ser JSON, View, arquivos, etc.

## Responsabilidades
- Não deve conter lógica de negócio
- Pode chamar serviços, repositórios e modelos
