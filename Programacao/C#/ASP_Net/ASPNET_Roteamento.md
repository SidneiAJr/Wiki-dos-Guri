# ASP.NET Core - Roteamento

## O que é Roteamento
Define como a aplicação responde a diferentes URLs.

## Formas de Roteamento
- Roteamento por Convenção
- Roteamento por Atributos

## Parâmetros de Rota
Permite capturar dados diretamente da URL.

## Rota Padrão MVC
Controllers → Ações → Parâmetros opcionais
