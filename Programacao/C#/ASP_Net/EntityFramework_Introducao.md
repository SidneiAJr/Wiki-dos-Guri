# Entity Framework Core - Introdução

## O que é
ORM (Object Relational Mapper) usado para interagir com bancos de dados usando classes ao invés de SQL manual.

## Conceitos
- DbContext → Gerencia conexão e modelos
- DbSet → Representação de tabelas
- Models → Representação de entidades do banco

## Vantagens
- Redução de SQL complexo
- Mapear tabelas para objetos
- Manutenção e leitura facilitadas
