# Entity Framework Core - Migrations

## O que são Migrations
Mecanismo para versionar e aplicar mudanças no banco de dados.

## Objetivo
- Atualizar tabelas conforme o código muda
- Evitar criação manual de scripts SQL

## Etapas
1. Criar Migration
2. Revisar alterações
3. Aplicar ao banco

## Benefício
Manter banco e código sempre sincronizados.
