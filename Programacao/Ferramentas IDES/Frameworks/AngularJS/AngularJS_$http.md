# AngularJS — $http

`$http` é o serviço utilizado para comunicação com servidores externos. Ele permite enviar e receber dados utilizando requisições web.

---

## Objetivo do `$http`
- Carregar dados de APIs externas
- Enviar informações para servidores
- Sincronizar dados entre front-end e back-end

---

## Métodos Comuns

| Método | Uso |
|-------|----|
| GET | Buscar informações |
| POST | Enviar dados |
| PUT | Atualizar dados existentes |
| DELETE | Remover informações |

---

## Visão Geral
`$http` possibilita que páginas AngularJS interajam com sistemas externos, tornando-as dinâmicas e conectadas.
