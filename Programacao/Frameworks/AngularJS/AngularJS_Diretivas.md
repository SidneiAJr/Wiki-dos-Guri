# AngularJS — Diretivas

Diretivas são instruções especiais que podem ser adicionadas ao HTML para estender seu comportamento. Elas permitem que o AngularJS controle elementos, atributos ou eventos de forma dinâmica.

A diretiva transforma o HTML em algo interativo e ligado à lógica da aplicação.

---

## Função das Diretivas
- Conectar a interface com a lógica da aplicação
- Controlar exibição e comportamento dos elementos
- Reagir a condições, listas, eventos e estados

---

## Diretivas Comuns

| Diretiva | Finalidade |
|--------|------------|
| `ng-app` | Inicia a aplicação AngularJS |
| `ng-model` | Conecta valores ao `$scope` |
| `ng-controller` | Define o controlador |
| `ng-repeat` | Repete elementos a partir de listas |
| `ng-if` | Exibe ou remove elementos conforme condição |
| `ng-show` | Mostra o elemento quando a condição é verdadeira |
| `ng-hide` | Esconde o elemento quando a condição é verdadeira |
| `ng-click` | Associa ações da interface às funções |

---

## Visão Geral
As diretivas permitem criar páginas dinâmicas sem manipular o DOM manualmente.
