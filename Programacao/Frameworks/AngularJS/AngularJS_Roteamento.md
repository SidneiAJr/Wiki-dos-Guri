# AngularJS — Roteamento

Roteamento permite transformar a aplicação em uma **Single Page Application (SPA)**, onde o conteúdo muda sem recarregar a página inteira.

---

## Objetivo do Roteamento
- Trocar telas sem recarregar o navegador
- Organizar a navegação da aplicação
- Associar URLs a páginas e controladores

---

## Conceitos Importantes

| Conceito | Significado |
|---------|-------------|
| Rota | Caminho da URL que leva a uma view |
| View | Conteúdo exibido para uma rota |
| Controller | Lógica associada à interface |
| Template | Estrutura do conteúdo da tela |

---

## Visão Geral
O roteamento permite criar experiências mais fluidas, rápidas e modernas no navegador.
