# Atalhos do NetBeans

## Atalhos Gerais
- **Ctrl + Shift + N**: Criar um novo projeto
- **Ctrl + Shift + O**: Abrir um arquivo
- **Ctrl + Shift + F**: Organizar e formatar o código automaticamente
- **Ctrl + S**: Salvar o arquivo atual
- **Ctrl + Shift + S**: Salvar todos os arquivos
- **Ctrl + Z**: Desfazer
- **Ctrl + Y**: Refazer
- **Ctrl + F4**: Fechar a aba do arquivo atual
- **Ctrl + F1**: Abrir a ajuda do NetBeans

## Atalhos de Navegação
- **Ctrl + Tab**: Alternar entre os arquivos abertos
- **Ctrl + Shift + T**: Navegar até a classe ou método
- **Alt + F1**: Mostrar a janela de navegação do projeto
- **Ctrl + Shift + F12**: Maximizar ou minimizar a janela do editor de código
- **Ctrl + 1**: Ir para a barra de ferramentas
- **Ctrl + Shift + L**: Ir para a linha especificada no arquivo

## Atalhos de Execução
- **F6**: Executar o projeto
- **Shift + F6**: Executar a classe com o método `main` aberto
- **Ctrl + F11**: Alternar entre depuração e execução
- **Ctrl + F5**: Executar com depuração

## Atalhos de Edição de Código
- **Ctrl + Space**: Autocompletar código (como `System.out.println`)
- **Ctrl + Shift + Space**: Exibir a lista de parâmetros de um método
- **Ctrl + D**: Excluir a linha atual
- **Ctrl + E**: Fechar a janela de editor atual
- **Ctrl + R**: Substituir texto
- **Ctrl + Shift + J**: Comentar/descomentar o bloco de código selecionado
- **Alt + Shift + F**: Organizar o código (formatar o código do projeto)
- **Ctrl + Shift + I**: Reformatar o código (indentação)

## Atalhos de Depuração
- **F5**: Iniciar ou continuar a execução com depuração
- **F7**: Entrar no método durante a depuração (Step Into)
- **F8**: Avançar para a próxima linha (Step Over)
- **Shift + F8**: Sair do método atual (Step Out)
- **Ctrl + F2**: Parar a depuração

## Atalhos de Refatoração
- **Alt + Shift + R**: Renomear a variável, método ou classe
- **Ctrl + Alt + Shift + T**: Refatorar para extrair método
- **Ctrl + Alt + M**: Refatorar para mover métodos

## Atalhos de JUnit
- **Ctrl + Shift + F10**: Executar o JUnit Test atual
- **Shift + F10**: Executar o projeto de teste JUnit atual
