# AngularJS — Filtros

Filtros são utilizados para modificar a forma como valores são exibidos no HTML. Eles não alteram os dados, apenas sua apresentação visual.

---

## Aplicações dos Filtros
- Formatação de textos
- Conversão de letras
- Ajuste de moedas e números
- Organização de listas

---

## Filtros Comuns

| Filtro | Função |
|--------|--------|
| `uppercase` | Converte texto para letras maiúsculas |
| `lowercase` | Converte texto para letras minúsculas |
| `currency` | Formata valores monetários |
| `date` | Formata datas |
| `number` | Ajusta quantidade de casas decimais |
| `filter` | Filtra elementos de uma lista |
| `orderBy` | Ordena listas |

---

## Visão Geral
Filtros ajudam a tornar a exibição de dados mais clara e organizada, sem alterar o conteúdo original.
