# Linux | Softwares que Uso

No meu dia a dia com Linux, costumo usar os seguintes programas e ferramentas:

- **Vim** — editor de texto/ código
- **htop** — monitor de processos
- **gpart** — ferramenta de partições
- **Bash** — shell e scripts
- **VS Code** — editor de código moderno
- **Python** — linguagem de programação
- **Firefox** — navegador
- **VLC** — player de mídia
- **LibreOffice** — suíte de escritório
- **Nmap** — ferramenta de rede e segurança
